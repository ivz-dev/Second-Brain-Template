## Здоров'я та спорт

### 💪 Тренер (Бодібілдінг)
**Файл:** `skills/trainer.md`
**Активація:**
```
як тренер, [запит]
поглянь на мою програму тренувань
склади програму на гіпертрофію / подорож / тиждень без залу
```
*Стиль:* evidence-based, конкретні числа, Jeff Nippard / Mike Israetel підхід

---

### 🥗 Нутриціолог
**Файл:** `skills/nutritionist.md`
**Активація:**
```
як нутриціолог, [запит]
проаналізуй моє харчування за тиждень
що їсти під час подорожі / на дефіциті / перед змаганнями
```
*Стиль:* Layne Norton / Peter Attia підхід, завжди цифри

---

### 🧠 Спортивний психолог
**Файл:** `skills/sports_psychologist.md`
**Активація:**
```
як спортивний психолог, [запит]
немає мотивації тренуватись
відчуваю вигорання / готуюсь до старту
```
*Стиль:* ACT + Flow state, конкретні інструменти без теорії
