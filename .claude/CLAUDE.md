# Second Brain — Інструкції для Claude

Це персональна база знань Igor Zakutynsky (CTO, науковець, спортсмен, мандрівник).

## Контекст
- Власник — CTO + науковець + спортсмен + мандрівник
- Мова нотаток: українська (але англійські терміни допустимі)
- Система: PARA + Daily Notes + Markdown + Obsidian

## Структура
- `00_Inbox/` — необроблені захоплення
- `01_Daily/` — щоденні нотатки (YYYY-MM-DD.md)
- `02_Projects/` — активні проекти (Work/, Science/)
- `03_Areas/` — постійні сфери (CTO/, Research/, Health/, Travel/)
- `04_Knowledge/` — база знань (Books/, Videos/, Ideas/, Concepts/)
- `05_Archive/` — архів
- `_Templates/` — шаблони

## Як допомагати

### Обробка Inbox
Коли просять обробити Inbox:
1. Прочитай кожен файл в `00_Inbox/`
2. Визнач тип (нотатка / ідея / задача / посилання)
3. Запропонуй конкретний шлях куди перемістити
4. Запитай підтвердження перед переміщенням

### Тижневий огляд
Коли просять тижневий огляд:
1. Знайди всі Daily нотатки за тиждень
2. Витягни незакриті задачі (- [ ])
3. Підсумуй тренування (середні показники)
4. Підсумуй зустрічі та рішення
5. Знайди невиконані Action Items

### Аналіз здоров'я
Коли просять аналіз спорту/харчування:
1. Знайди всі Training логи за вказаний період
2. Підрахуй: кількість тренувань, загальний час, розподіл по видах
3. Якщо є харчування — порівняй з цілями
4. Якщо є БАДи — перевір регулярність прийому

### Нові нотатки
Коли просять створити нотатку:
1. Використай відповідний шаблон з `_Templates/`
2. Заповни frontmatter
3. Помісти у правильну папку
4. Запитай чи додати посилання на пов'язані нотатки

## Скіли (Advisor Personas)

Повний список скілів і як їх активувати — у файлі `.claude/SKILLS.md`.

Якщо запит починається з "як [тренер / нутриціолог / борд / ...]":
1. Прочитай відповідний файл із `.claude/skills/`
2. Увійди в роль і відповідай у стилі цього скілу
3. Читай релевантні нотатки з second brain перед відповіддю

**Доступні скіли:**
- `skills/trainer.md` — тренер (бодібілдінг)
- `skills/nutritionist.md` — нутриціолог
- `skills/sports_psychologist.md` — спортивний психолог
- `skills/travel_manager.md` — тревел менеджер
- `skills/science_board.md` — борд науковців (CS, IoT, High Load)
- `skills/tech_board.md` — борд менеджерів (Henderson, Torvalds, Collison, Hashimoto...)
- `skills/one_on_one.md` — менеджерські скіли, аналізатор 1-on-1

## Правила
- Не видаляй нотатки без явного підтвердження
- Зберігай мову оригіналу нотатки
- При переміщенні файлів завжди показуй звідки → куди
- Frontmatter — обов'язковий для всіх нотаток
